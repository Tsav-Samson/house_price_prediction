About the Software Cost Prediction Application

A PhD Thesis by Mrs. Beatrice Akumba

The effort and duration of a software can not be accurately ascertained by a one-side fits all approach, but an estimation can be made by using certain features and deploying a comparative approach.

  • Different machine learning methods are compared for best performance in making predictions
  • Predicting system is developed using two machine learning approaches both instances of the linear regression class in python's scikit learn , which are used to predict the software effort and duration, and afterwards using the kalman filter to get a near accurate estimate of both respectively.
